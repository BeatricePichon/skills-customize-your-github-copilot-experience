# 📘 Assignment: Python Data Visualization

## 🎯 Objective

In this assignment, you will create charts and graphs in Python to communicate insights from data. You will practice selecting appropriate chart types, building visualizations with either matplotlib or plotly, and adding clear labels and titles.

## 📝 Tasks

### 🛠️ Build Core Visualizations

#### Description
Use the provided starter data to create multiple chart types that represent different parts of the dataset.

#### Requirements
Completed program should:

- Create at least 3 different charts (for example: line chart, bar chart, and scatter plot).
- Use either matplotlib or plotly consistently in the program.
- Save each chart as an image file in the assignment folder.
- Include chart titles and axis labels for every visualization.


### 🛠️ Interpret and Present Insights

#### Description
Add a short analysis section in your program output or comments that explains what each chart shows and why that chart type is appropriate.

#### Requirements
Completed program should:

- Print or include at least 2 observations based on trends, comparisons, or outliers in the data.
- Match each observation to a specific chart.
- Use readable colors, marker sizes, or styles so visual elements are easy to interpret.
- Organize code into clear steps (load data, chart creation, interpretation, save/export).
